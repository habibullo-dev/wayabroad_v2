# Coursework (not part of the WayAbroad app)

These are entrepreneurship/business-class reference materials (lecture PDFs and a class draft)
that happened to live in the project folder. They were relocated here to keep the repo root clean.

**They are intentionally git-ignored** (see the repo `.gitignore`) — they're large binaries
unrelated to the application, so they stay on disk but are not version-controlled. This `README.md`
is the only tracked file in this folder.

If these should actually be version-controlled (or removed), let the maintainer know.

Contents (as relocated 2026-05-30):

- `Chapter 1–11 *.pdf`, `Ch 10 - Funding (2025).pdf` — course lecture chapters
- `Guidelines for writing a business plan (Spring 2026).pdf`
- `The_Entrepreneurs_Financial_Toolkit *.pdf`
- `Technology Innovation Class Draft.docx`

The WayAbroad project docs (development plan, build brief, business plan) live one level up in `docs/`.
